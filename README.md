# Rick and Morty Character Explorer

Here's my rick and morty char gen up and running. Please input a number between 1 and 826 and press enter.: https://rickandmortyapishowcaase.vercel.app/

## December 2025 Upgrade

Upgraded this school project with new features:

- **Character Search by Name** - Search for characters by name instead of just ID numbers
- **Random Character Button** - Discover random characters from the Rick and Morty universe
- **Episodes Display** - View all episodes a character appears in
- **Location Details** - Click on origin or location to see full location information
- **Status Filters** - Filter by Alive, Dead, or Unknown status (shows random character each time)
- **Animated Space Background** - Beautiful starry sky with flying saucers in the sidebars

Built using the [Rick and Morty API](https://rickandmortyapi.com/).
